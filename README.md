## About

A fast, responsive personal portfolio / landing page built with pure HTML, CSS and JavaScript. No frameworks, no build tools — just clean, hand-crafted code that loads instantly and looks great on every screen.

## 1-Click Setup

```bash
# Clone the repo
git clone https://github.com/kakadeaarush1-svg/portfolio-website.git

# Open in browser (no server needed)
open portfolio-website/index.html

# Or use a local server
npx serve portfolio-website
```

Or just fork the repo and enable **GitHub Pages** in Settings → Pages → Source: `main` branch.

---

## Build Process / Learnings

- Designed the gradient palette using CSS custom properties for easy theming
- Wrote the layout with CSS Flexbox — no grid framework needed
- Used `transition: all 0.3s` on links for smooth hover states
- Kept JS minimal: only nav toggle logic for mobile
- Deployed continuously via GitHub Actions (`.github/workflows/pages.yml`)
